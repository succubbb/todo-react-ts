Это простое приложение TODO для отслеживания задач. Использовался React + TypeScript + Tailwind CSS.
Функционал: 
- добавление задачи
- удаление задачи
- отметка выполнено/не выполнено
- счетчик задач

Также планируется добавление пагинации и выпадающего меню с редактированием задачи, дублированием и удалением.

Основывалась на статье https://vk.com/away.php?to=https%3A%2F%2Ftypeofnan.dev%2Fyour-first-react-typescript-project-todo-app%2F&utf=1
